
Gotcha


Автор: [https://github.com/hedromanie]

ОПИСАНИЕ
--------
Gotcha – это программа для тестирования устойчивости сетевых устройств и протоколов
в контролируемых лабораторных условиях. Она позволяет генерировать различные типы
сетевого трафика, имитировать атаки (DHCP starvation, ARP spoofing, DoS и др.)
и перехватывать пакеты для анализа.

ТРЕБОВАНИЯ К СИСТЕМЕ
--------------------
- ОС: Windows 10/11 (64-bit)
- Права администратора
- Установленный Npcap

УСТАНОВКА
---------
1. Откройте установщик из последнего релиза
2. Пройдите процедуру установки
3. Запустите программу
4. Прочитайте руководство пользователя guide.html

Рекомендованное ПО
------------------
Npcap (Обязательно) — драйвер для захвата и генерации пакетов. Скачайте с https://npcap.com. При установке обязательно выберите опцию WinPcap API-compatible Mode
Wireshark — для анализа трафика и отладки. Скачайте с https://wireshark.com
Nmap — для нахождения уязвимостей. Скачайте с https://nmap.com
TCP optimizer — для оптимизации сетевой карты ( возрастет кол-во создаваемых пакетов для dos атаки ). Скачайте с https://speedguide.net
Microsoft Visual C++ Redistributable (последняя версия)
